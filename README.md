# Loan Approval Prediction System

## 📌 Project Overview

This project is a **machine learning-based loan approval system** designed to automate the credit risk assessment process. It predicts whether a loan should be approved or rejected based on applicant details such as income, employment status, credit history, and loan amount.

The goal is to move beyond simple “Yes/No” predictions and provide **data-driven insights** that help in better decision-making.

---

## 🎯 Objectives

* Automate loan approval prediction using Machine Learning
* Reduce human bias in financial decisions
* Identify high-risk applicants efficiently
* Provide interpretable insights for better decision-making

---

## 🧠 Machine Learning Models Used

* **Logistic Regression** → Baseline model for comparison
* **Decision Tree Classifier** → Captures non-linear relationships
* **Random Forest Classifier** → Final model for better accuracy

---

## ⚙️ Technical Workflow

### 1. Data Preprocessing

* Handling missing values using imputation
* Encoding categorical features (Label Encoding)
* Feature selection and scaling

### 2. Model Training

* Train-test split
* Model training on multiple algorithms
* Performance comparison

### 3. Model Evaluation

* Accuracy Score
* Confusion Matrix
* Model selection based on performance

### 4. Deployment (Streamlit App)

* Built interactive UI using Streamlit
* Real-time predictions from user input
* Clean dashboard for usability
